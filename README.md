# SPPU TE DSBDA Practicals

This repository contains DSBDA (Data Science and Big Data Analytics) practical assignments for Savitribai Phule Pune University (SPPU) TE Computer Engineering students.

It includes practical programs, Python implementations, machine learning algorithms, data visualization, big data processing, and reference solutions based on the SPPU syllabus.

## Contents

- Data preprocessing
- Data visualization
- Regression
- Classification
- Clustering
- KNN
- Decision Tree
- Naive Bayes
- MapReduce
- Big data processing
- Practical assignments

## University

Savitribai Phule Pune University (SPPU)

## Branch

Third Year Engineering (TE) Computer Engineering

## Subject

Data Science and Big Data Analytics (DSBDA)

## Purpose

This repository is created for SPPU students to access DSBDA practical programs, source code, lab assignments, and practical exam preparation resources.

These practical programs are shared for educational purposes and learning.
